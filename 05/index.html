About

NTT Tech Conference は、NTT グループのソフトウェアエンジニアたちが一堂に会し、NTT グループ内外のエンジニアたちと技術交流を行うためのカンファレンスです。
NTT グループ内には各種 OSS のコミッタ、メンテナ、コントリビュータをはじめとしたエンジニアや、各社の様々な案件でシステム開発を行うエンジニアがおり、本イベントではそれらのエンジニアがもつノウハウや悩みを NTT グループ内外のエンジニアと共有・議論することで、参加者がもっとおもしろいエンジニアになることを目的としています。

  • 名称：NTT Tech Conference #5
  • 日時：2021年2月26日（金）13:30〜17:30
  • 会場：オンライン
  • 主催：NTT Developers
  • 参加費：無料（事前登録制）

IP 通信では動画、メール、Web など、様々な情報がやり取りされています。
障害発生の深堀作業においてアプリレイヤーで解決できなかった場合、パケットを真心こめて作成しなければならない局面に遭遇することが極めてまれに起こります。
このセッションでは、真心こめてパケットを作成するツール類を紹介を行うことに加え、私たち開発者に親和性の高いパケット生成ツールの試用する段階までを紹介します。

学べること

  • 【前編】Windows 10（GUI 環境）でのパケット生成ツール利用
  • 【後編】Linux 環境（CLI 環境）でのパケット生成ツール利用

必要なもの

  • 【前編】Windows 10（事前に Wireshark・Win10Pcap・ipsendwin・Colasoft Packet Builder をインストールしておいてください）
  • 【後編】Docker をインストールした端末を用意し、事前に次のコマンドでイメージのダウンロードをお願いします
    docker pull jwat/ubuntu2004:latest

PDF のコピペが文字化けするのはなぜか？ 〜 CID / GID と原ノ味フォント 〜

14:00 〜 14:25

中級
Application
OSS

PDF の中には「見」や「高」などの文字が検索できなかったり、コピペすると似たような別の字に化けたりするものが結構あるのをご存じでしょうか。
住所を集計しようとしても長野の「長」や埼玉の「玉」がなどが化けてうまくいかなかったり、様々な文字で影響が出るもので、康熙部首・CJK 部首補助の文字化け問題と言われます。
もちろん、これらの文字を使っていても化けない PDF もあります。
しかし、広く使われている PDF 作成ツールとフォントで化ける場合があり、多くの PDF がこの問題を抱えています。
本セッションでは、なぜ化けるのか PDF やフォントの仕組みから紐解きます。
そして、化けない PDF が作成できる日本語 TeX について、その対策とデフォルト和文フォントの原ノ味フォントを紹介します。

学べること

PDF やフォントの仕組み。

鈴ヶ嶺 聡哲（NTT コミュニケーションズ）

macOS の仮想化技術について 〜 virtualization-rs Rust bindings for Virtualization.framework 〜

15:00 〜 15:25

中級
Container
OSS
Virtualization / Hypervisor

本セッションではまず仮想化技術についての基本的な背景をご説明し、macOS の仮想化技術の変遷や仮想化技術を支えるツールについてご紹介します。
特に注目する技術として、macOS 11 Big Sur から Intel ベースの Macや Apple Silicon ベースの Mac で VM を作成する Virtualization.framework という高レベル API が追加されました。
Docker などは内部で低レベル API の Hypervisor.framework を使用した HyperKit を Virtualization.framework に置き換えつつあります。
最後に、この Virtualization.framework を Rust で binding するライブラリを作成したので、そのご紹介と Linux を boot するデモをしたいと思います。

学べること

  • 仮想化技術・ハイパーバイザの基本について
  • macOS の仮想化技術の変遷について
  • macOS の仮想化を支える Hypervisor.framework, Virtualization.framework などの API や xhyve, HyperKit などのツールについて
  • Rust で作る macOS アプリケーションについて
